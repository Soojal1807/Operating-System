# OS Concepts Playground 

This is my personal repository for learning operating system (OS) concepts. I am building simple, hands-on projects to understand how things work under the hood. All code is written in **C++**. 

## What’s Inside 
* **Memory Management:** Simulating page replacement algorithms (FIFO, LRU).
* **Process Scheduling:** Implementing basic CPU schedulers like FCFS and Round Robin.
* **Concurrency:** Exploring threads, mutexes, and deadlocks.

## Getting Started 
You only need a modern C++ compiler (like `g++`) to run these projects. 
1. **Clone the repo:** `git clone <repository-url>`
2. **Compile:** `g++ -o filename filename.cpp`
3. **Run:** `./filename`

## Learning Goals 
* Understand how the OS  manages resources.
* Improve my **C++**  skills by applying them to real-world system logic.
* Learn to debug memory leaks and race conditions.

*Note: This is a self-study space and not a professional or so called "production-grade" repository. Code might be messy or all over the place, Feel free to peek around or open an issue if you spot a bug.*
